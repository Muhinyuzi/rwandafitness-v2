# RwandaFitness — Style Guide

## Brand Colors

Primary:
```text
Turquoise / Cyan
```

Accent:
```text
Orange
```

Neutral:
```text
Zinc / Slate
```

## UI Principles

- Clean cards
- Rounded corners
- Soft shadows
- Clear call-to-action buttons
- Mobile-first layout
- Strong visual hierarchy

## Typography

Use modern sans-serif typography.

## Components

Common components:
- Navbar
- Footer
- Cards
- Buttons
- Forms
- Status badges

## Naming

### Frontend
- Components: `PascalCase`
- Variables: `camelCase`
- Routes: lowercase with hyphens

### Backend
- Models: `PascalCase`
- Fields: `snake_case`
- API paths: lowercase with hyphens where needed

## Tone

RwandaFitness should sound:
- motivational
- simple
- modern
- trustworthy
- community-focused
